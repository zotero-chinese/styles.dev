    注意：使用author+year 样式的时，如果中文两个作者的和前面有逗号(,)，如显示为（金红兰, 和金龙勋, 2021），请确认条目的作者是是否经过合并操作，如果合并过请拆分(可以使用茉莉花插件https://github.com/l0o0/jasminumopen in new window)。
